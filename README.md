# Projeto Landing Page - Catálogo de Carros

Este repositório contém o desenvolvimento de uma interface para exibição e venda de veículos, focada em organização de layout e responsividade.

## O que foi realizado:
* **Estruturação de Vitrine:** Criação de uma grade de produtos para exibir modelos de carros com suas especificações.
* **Layout Responsivo:** Implementação de CSS para que o site se adapte corretamente a diferentes tamanhos de tela.
* **Design de Interface:** Aplicação de estilos modernos para criar uma experiência de navegação profissional e intuitiva.
* **Semântica e Organização:** Uso de tags HTML5 para garantir uma estrutura de código limpa e acessível.

## Tecnologias:
* HTML5
* CSS3
* Git / GitHub
